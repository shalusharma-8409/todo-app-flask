# 🚀 To-Do App (Flask)

A simple and interactive To-Do Web App built using Flask.

---

## ✨ Features

- Add tasks
- Delete tasks
- Mark tasks as complete ✔
- Track daily progress 📊
- Search tasks 🔍
- Add motivational quotes 💬

---

## 🛠 Tech Used

- Python (Flask)
- HTML
- CSS
- JSON

---

## 🚀 How to Run

1. Install Flask:
pip install flask

2. Run app:
python app.py

3. Open browser:
http://127.0.0.1:5000
